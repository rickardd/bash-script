# VocabMaster Bash Script

A simple and silly command line vocabulary trainer

Language codes
- en-sv - Swedish
- en-sp - Spanish
- en-mao - Maori

``bash cd /path/to/file/```

``. index.sh [language code]``

or...

``bash cd /path/to/file/```

``. add-alias.sh``

Now run following anywhere. 

``vocab-master [language code]``
